# 🚀 Clarity: The AI Assistant for Teams

Clarity is a helpful tool that makes sure everyone on a team is on the same page. Sometimes, a leader says one thing, but the team thinks something else. Clarity uses AI to check those ideas and give the team a "Clarity Score" so they can work better together.

## 🛠️ How It Was Built

- **The Look:** Built with React, making it fast and easy to use on a phone or computer.
- **The Brain:** Powered by Node.js and Google Gemini AI, which reads the team's goals and gives helpful advice.
- **The Storage:** Uses Google Firebase to safely save team goals and scores.
- **The Payments:** Connected to Stripe so leaders can easily upgrade to the "Pro" version.
- **The Home:** Hosted on Google Cloud, which keeps the app running 24/7 on the internet.

## 🏗️ Smart Features

### 🔐 Keeping Secrets Safe
I built this app using a "professional first" approach. All the important keys (like the ones for AI and Payments) are hidden on a private server. This keeps the app safe from hackers and ensures that private information stays private.

### 💰 The 30-Check Limit
To help turn this into a real business, I added a simple rule:
- **Free Version:** Every team gets 30 free checks. This is enough for a team to see how great the app is.
- **Pro Version:** If they need more, they can click one button to pay $49 through Stripe. Once they pay, the limit is removed instantly.
